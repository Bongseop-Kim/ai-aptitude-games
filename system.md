# AI 역량 검사 대비 앱 – 훈련용 게임 정리표

> **목표**  
> 기업 AI 역량 검사 대비용 앱에 바로 적용 가능한 검사/게임만 선별  
> (게임화 가능 · 자동 채점 · 반복 훈련 효과 중심)

---

## 검사 · 게임 유형별 정리

| 검사 / 게임 유형        | 측정 역량                 | 훈련 난이도 | 진행 방법                                                              |
| ----------------------- | ------------------------- | ----------- | ---------------------------------------------------------------------- |
| 가위바위보 (규칙 전환)  | 규칙 전환, 억제, 반응속도 | 하          | 제시된 관점(나/상대)에 맞게 이기거나 지도록 선택. 라운드마다 규칙 전환 |
| 도형 회전 / 반전        | 공간지각, 정신 회전       | 중          | 도형을 회전·반전하여 목표 도형과 동일하게 만들기                       |
| 약속 정하기 (기억 게임) | 작업기억, 정보 갱신       | 중          | 3회 제시된 정보 중 공통(또는 미출현) 항목 기억 후 선택                 |
| 길 만들기               | 패턴 인식, 계획 능력      | 중          | 울타리를 배치해 사람–택시를 연결하는 경로 완성                         |
| 마법약 만들기 (확률)    | 확률 추론, 통계적 학습    | 중          | 조합별 색 출현 빈도를 기반으로 더 가능성 높은 색 선택                  |
| 숫자 누르기             | 선택적 주의, 작업기억     | 하          | 활성 숫자 클릭 / 조건(2번 클릭, 건너뛰기)에 맞춰 순서대로 입력         |
| 도형 순서 기억 (N-back) | 작업기억, 업데이트        | 중          | 현재 도형이 n번째 이전 도형과 같은지 판단 (2-back → 3-back)            |
| 고양이 술래잡기         | 메타인지, 기억 기반 판단  | 중          | 위치 겹침 여부 판단 후 확신도(정도)를 함께 선택                        |
| Mental Rotation (표준)  | 공간지각, 회전 추론       | 중          | 서로 같은 도형인지 회전 각도를 고려해 판단                             |
| Stroop Test             | 억제, 선택적 주의         | 하          | 단어 의미가 아닌 글자 색상 기준으로 빠르게 선택                        |
| Go / No-Go              | 충동 억제, 반응 통제      | 하          | 특정 자극에는 클릭, 특정 자극에는 클릭 금지                            |
| Task Switching          | 인지 유연성, 전환 비용    | 중          | 규칙 A/B를 번갈아 적용하여 동일 자극을 다르게 처리                     |
| Visual Search           | 선택적 주의, 시각 탐색    | 하          | 다수의 방해 자극 중 목표 자극을 빠르게 탐색                            |
| Probabilistic Learning  | 확률 학습, 보상 추론      | 중          | 보상 확률이 높은 선택지를 반복 시도하며 학습                           |

---

## 설계 메모 (중요)

- **훈련 난이도 기준**
  - 하: 규칙 단순, 반복 숙련용
  - 중: 전략·패턴 학습 필요 (연습 효과 큼)
- **메타인지(확신도)**는 단독 게임보다  
  → 고양이 술래잡기처럼 **기존 게임에 옵션으로 결합** 권장
- 위 구성만으로 **대기업 AI 역량 검사 유형 대부분 커버 가능**

---

## 추천 MVP 구성 (8개 내외)

- 도형 순서 기억 (N-back)
- 도형 회전
- Stroop
- Go / No-Go
- 가위바위보 (규칙 전환)
- 약속 정하기
- 숫자 누르기
- 마법약 만들기

---

다음 단계로 필요하면

- **MVP 탭 구조 (훈련 / 실전 / 리포트)**
- **게임별 점수화·랭킹·피드백 지표 설계**
- **유료화 가능한 훈련 패키지 분리안**  
  바로 이어서 정리 가능.
